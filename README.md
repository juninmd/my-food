> A modern meal planning and health application built with Flutter and Node.js. Orchestrated by **Jules-Architect** and **Chef-Domain**.

## ✨ Features

- **Personalized Meal Plans**: AI-driven dietary suggestions based on user goals.
- **Health Tracking**: Comprehensive dashboard for monitoring weight, nutrients, and progress.
- **Automated Releases**: Built with `semantic-release` for seamless versioning.
- **Recipe Management**: Store and organize healthy recipes with full nutritional data.
- **Interactive HUD**: Modern, responsive interface built with Flutter.

## 🛠️ Tech Stack

- **Backend**: Node.js + Express 5
- **Frontend**: Flutter (Dart)
- **CI/CD**: Semantic Release + GitHub Actions
- **State Management**: Provider / Riverpod

## 🚀 Getting Started

```bash
# Install dependencies
pnpm install

# Build release (Semantic)
pnpm run release
```

## 🛡️ Antigravity Protocol

This project adheres to the **Antigravity** code standards:
- **Clean Backend Architecture**: Express 5 patterns with separation between routes and business logic.
- **Modular Flutter Widgets**: Reusable components for a consistent visual identity.
- **Strict Typing**: Full typing for API contracts and Flutter models.
- **150-Line Limit**: Applied to both Node.js and Flutter logic files.
